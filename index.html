01 App Architecture

  • MVVM 기반 상태 관리와 ViewModel 테스트 설계
  • Coordinator로 화면 전환 책임 분리
  • Dependency Injection으로 교체 가능한 모듈 구성

02 UI Engineering

  • SwiftUI 선언형 UI와 UIKit 기반 레거시 화면 대응
  • Auto Layout, Dynamic Type, VoiceOver 고려
  • Apple HIG에 맞춘 계층, 여백, 인터랙션 정리

03 Data & Networking

  • URLSession, async/await, Combine 기반 API 연동
  • Core Data/SwiftData 로컬 저장소 모델링
  • 로딩, 빈 상태, 실패, 재시도 플로우 설계

04 Quality & Delivery

  • XCTest로 핵심 비즈니스 로직 검증
