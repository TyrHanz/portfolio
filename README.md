# TyrHanz iOS Developer Portfolio

Apple Glass 스타일로 제작한 iOS 개발자 포트폴리오입니다. GitHub Pages에 바로 올릴 수 있도록 외부 라이브러리 없이 `index.html` 단일 파일로 구성했습니다.

## Preview

- Portfolio page: `index.html`
- Target style: Apple Glass, project-first, Korean-first
- Main sections: Hero, Featured Projects, Skills, Growth Timeline, Contact

## Highlights

- Swift, SwiftUI, UIKit, MVVM 중심의 iOS 개발자 브랜딩
- 프로젝트마다 문제, 해결, 내 역할, 기술 스택, 결과를 빠르게 볼 수 있는 카드 구성
- 반응형 레이아웃, iPhone 목업, glassmorphism UI, scroll reveal 모션
- 접근성 기본값 포함: semantic section, skip link, focus state, reduced motion 대응

## GitHub Pages Setup

1. Repository `Settings`로 이동합니다.
2. `Pages` 메뉴를 선택합니다.
3. `Build and deployment`에서 `Deploy from a branch`를 선택합니다.
4. Branch는 `main`, folder는 `/ (root)`로 설정합니다.
5. 저장 후 배포 URL이 생성될 때까지 기다립니다.

## Customize

`index.html`에서 아래 값을 실제 정보로 교체하면 됩니다.

- `TyrHanz`를 실명 또는 표시명으로 변경
- `your.email@example.com`을 실제 이메일로 변경
- `your-linkedin`을 실제 LinkedIn slug로 변경
- `project-one`, `project-two`, `project-three` 링크와 설명을 실제 프로젝트로 변경
